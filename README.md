# Embedded Systems Labs

This is a compilation of the labs completed for the Embedded Systems Course (Fall 2025).

## Lab 1 – Introduction & Toolchain Setup
**Objective:** Become familiar with the development environment and foundational embedded concepts.  
**Summary:** Set up the compiler, debugger, and project structure needed for ARM-based embedded development.

## Lab 2 – GPIO and Digital I/O
**Objective:** Learn how to configure and control General-Purpose Input/Output pins.  
**Summary:** Implemented basic LED control and switch inputs while exploring register-level programming.

## Lab 3 – Timers and Interrupts
**Objective:** Understand hardware timers and how interrupts handle asynchronous events.  
**Summary:** Configured periodic timers and wrote interrupt service routines to respond to hardware events.

## Lab 4 – Communication Protocols
**Objective:** Explore communication interfaces used in embedded devices.  
**Summary:** Implemented and tested basic communication using protocols such as UART, SPI, or I²C depending on the lab requirements.

## Lab 5 – ADC/DAC and Sensor Input
**Objective:** Interface with analog hardware using the microcontroller's ADC/DAC modules.  
**Summary:** Wrote code to sample analog signals, process them, and generate simple analog outputs.

## Lab 6 – Scheduling / RTOS Concepts
**Objective:** Introduce basic task scheduling and real-time operating system principles.  
**Summary:** Created a simple cooperative scheduler or worked with RTOS primitives such as tasks, delays, and synchronization.

## Lab 7 – Total Integration
**Objective:** Combine all prior concepts into a functional embedded system.  
**Summary:** Integrated GPIO, timers, interrupts, communication, and sensor inputs into a cohesive final lab.
